

# Рендеринг Markdown в чат-виджете

## Проблема
AI-консультант отвечает с markdown-форматированием (например, `**1500 проектов**`), но чат отображает сырой текст — пользователь видит звёздочки вместо жирного шрифта.

## Решение
Заменить простой `{m.content}` на компонент `ReactMarkdown` из уже установленной библиотеки `react-markdown`.

## Технические детали

### `src/components/ChatWidget.tsx`
1. Добавить импорт: `import ReactMarkdown from 'react-markdown'`
2. В рендере сообщений (строка ~114) заменить `{m.content}` на:
   ```tsx
   <ReactMarkdown className="prose prose-sm prose-invert max-w-none [&>p]:m-0">
     {m.content}
   </ReactMarkdown>
   ```
   - Для сообщений пользователя (`role === 'user'`) — оставить `prose-invert` (белый текст на синем фоне)
   - Для сообщений ассистента (`role === 'assistant'`) — использовать `prose prose-sm` без invert (тёмный текст на сером фоне)
   - `[&>p]:m-0` убирает лишние отступы параграфов внутри пузыря

Библиотека `react-markdown` уже установлена в проекте, новых зависимостей не нужно.
